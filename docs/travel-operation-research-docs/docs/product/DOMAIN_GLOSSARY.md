# Travel Operation Domain Glossary

Use these definitions consistently across database models, APIs, UI labels, reports, documentation, and team communication.

# 4. Travel-Domain Definitions

| Term | Meaning in the platform |
|---|---|
| FIT | Flexible or independent traveler booking, usually custom-built |
| GIT | Group inclusive tour with shared itinerary and arranged capacity |
| Series departure | Repeated package with fixed dates and capacity |
| Package | Multiple travel services sold together |
| Itinerary | Ordered travel plan containing dates, places, services, and activities |
| PNR | Passenger Name Record or equivalent reservation locator |
| TTL | Ticketing or payment time limit |
| Voucher | Proof/instruction for a supplier to deliver a service |
| Ground handler | Destination-side operator delivering transfers, guides, tours, and local services |
| Net rate | Supplier price before agency markup |
| Published rate | Public-facing or supplier-listed price |
| Markup | Amount or percentage added to cost |
| Commission | Income paid by a supplier or allocated to an agent/staff member |
| Reissue | Replacement of a ticket or service after a change |
| Refund | Return of eligible value after deductions |
| Void | Reversal within an allowed issuance window |
| No-show | Passenger did not use the reserved service |
| Allotment | Capacity held by contract for future sale |
| Release period | Date by which unused allotment must be returned |
| Rooming list | Group accommodation assignment list |
| Manifest | Operational passenger list |
| DMC | Destination management company |
| MICE | Meetings, incentives, conferences, and exhibitions |
| SIT | Special-interest tour |
| FAM | Familiarization trip used for product research and supplier development |
| Pax | Passenger or participant count |
| Service order | One fulfillment unit inside a booking, such as flight, hotel, visa, transfer, or tour |

---

## Additional Platform Terms

| Term | Platform meaning |
|---|---|
| Client | Person, family, company, institution, or agent responsible for the commercial relationship |
| Traveler / Passenger | Person consuming a travel service; may differ from the payer or client |
| Payer | Party responsible for payment |
| Service Order | One fulfillment unit inside a booking, such as air, hotel, transfer, visa, tour, insurance, cruise, rail, or miscellaneous service |
| Supplier Reservation | Reservation or request sent to an external supplier for one service order |
| Booking Readiness | Measured state showing whether payment, documents, confirmations, visas, tickets, vouchers, and operational preparation are complete |
| Expected Cost | Cost estimated during quotation or booking planning |
| Confirmed Cost | Cost accepted or confirmed by the supplier |
| Actual Cost | Final cost invoiced or settled |
| Expected Profit | Forecast profit before all costs are confirmed |
| Realized Profit | Final recognized profit after supplier settlement, commissions, refunds, fees, and currency variance |
| AR | Accounts receivable: money owed by clients or agents |
| AP | Accounts payable: money owed to suppliers |
| Maker-checker | Control requiring a different person to approve a sensitive transaction |
| SLA | Service-level deadline for response or completion |
| DMC | Destination management company |
| OTA | Online travel agency |
| NDC | Airline distribution standard used for modern airline offers and orders |
| GDS | Global distribution system used for travel inventory and reservation workflows |
| Service Recovery | Operational and financial action taken after a service failure |
| Readiness Gate | Mandatory condition that blocks progression until satisfied |
| Snapshot | Historical copy of cost, rate, terms, exchange rate, or policy used at the moment of sale or approval |
